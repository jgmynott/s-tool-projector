How the projection is built

1
Learn the stock's own dynamics
We pull ~2 years of daily closes, estimate annualized drift (μ) and volatility (σ), and fit a mean-reversion anchor by regressing price on its 1-year moving average.
2
Scale volatility to the current regime
σ is multiplied by a VIX-regime factor and further widened when earnings are within 30 days — the one systematic adjustment that empirically improves band calibration.
3
Simulate 2,000 forward paths
Each path is a blend of a Monte Carlo random-walk (30%) and a mean-reverting Ornstein-Uhlenbeck process pulling toward the MR target (70%). Neither dominates — the blend prevents both "infinite trend" and "forced reversion" failure modes.
4
Report percentiles, not point forecasts
P10 / P25 / P50 / P75 / P90 of the simulated terminal prices. The band is the answer — not a single number. If someone tells you an exact 1-year price target, they're selling, not forecasting.
What the model does NOT do. It doesn't read news, Reddit, or analyst reports. It doesn't apply crowd-sentiment or macro "tilts" — we backtested those through 2025 and they failed to improve accuracy. It doesn't output buy/sell/hold signals, because calibrated uncertainty and a single directional call are different (and usually conflicting) products.
